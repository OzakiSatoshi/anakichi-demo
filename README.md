# あな吉手帳術 Web アプリ（デモ公開用）

紙＋付箋の「あな吉手帳術」をデジタル化する Web アプリの**プロトタイプ（デモ）**です。
画面と操作感の確認用に公開しています。

- 公開URL（GitHub Pages）：https://ozakisatoshi.github.io/anakichi-demo/

## 使い方

1. 上記URLを開くと**ログイン画面**が出ます。デモ用なので、メールアドレス・パスワードは**どんな値でも入れます**（例：`demo@example.com` / `demo1234`）。
2. 画面下のナビ（**置／月／週／日**）で、置き場・マンスリー・ウィークリー・デイリーを切り替えられます。
3. ヘッダーの **AI** は音声入力からの付箋化（デモ・モック。提案のみで最終判断は人が行います）。

> 📱 スマホ優先のアプリです。「貼る・剥がす・動かす」の操作感はスマホ実機での確認がおすすめです。
> ⚠️ これはデザインと操作感の確認用プロトタイプです。再読み込みするとサンプルデータに戻ります（保存機能は今後実装）。
